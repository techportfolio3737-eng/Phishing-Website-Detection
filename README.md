# 🛡️ Phishing Website Detection using Machine Learning

## 📌 Overview
This project detects whether a website URL is **Phishing** or **Legitimate** using Machine Learning.

## 🎯 Features
- URL Feature Analysis
- Data Preprocessing
- Random Forest Classifier
- Model Training & Testing
- Model Saving using Joblib
- High Accuracy Prediction

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Joblib

## 📂 Project Files
- Phishing-Website-Detection.ipynb
- phishing_model.pkl
- requirements.txt
- README.md

## 📊 Results
- Model: Random Forest
- Accuracy: **100%**
